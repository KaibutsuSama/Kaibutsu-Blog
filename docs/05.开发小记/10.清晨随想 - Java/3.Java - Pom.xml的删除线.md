## 前言

使用软件：Idea

在开发的过程中，因为对父子级工程的一些操作不熟悉删除模块，再创建，就发现，为什么我的`Pom.xml`出现了一条删除线？？？

## 解决方案

1. 打开`Setting`

2. 搜索`maven` 或者在 `Build, Execution, Deployment` 找到 `Build Tools` 的 `maven`

3. 在`Ignored Files`把指定文件的 ✔ 给去掉

4. 解决问题

5. 还解决不了就自杀
